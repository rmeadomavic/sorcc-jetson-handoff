Section 04 · 15-60 min each

Apps

Three optional apps. Pick what you want. They are independent, so installing one does not require any of the others. Hydra Detect is what you used in class; Ollama gives you a private ChatGPT; ComfyUI does local image generation.

Before you install anything

Run preflight first. If it fails, fix the listed issues before continuing. Most blockers live in Section 02.

Shell · sorcc@jetson
$ bash scripts/preflight.sh
Memory budget reality check The Jetson Orin Nano Super has 8 GB of shared CPU/GPU RAM. After Linux + the Docker daemon (~1 to 1.5 GB), that's the budget for whatever you're running. You can comfortably run one of these at a time. Trying to run Hydra plus ComfyUI plus a 7B Ollama model concurrently will swap to disk and become unusably slow. Stop containers you're not actively using (docker stop hydra-detect) and you'll be fine.

What "idempotent" means here

Every install.sh in this section is safe to re-run. If the app is already installed, the script will detect it, check for updates if relevant, re-apply any config it manages, and not break your existing data. So when in doubt, re-run the installer rather than manually patching.

Where Docker images live

By default Docker stores images at /var/lib/docker/. On a 64 GB SD card with multiple apps installed, you'll fill it up. To move Docker storage to an external SSD, see the NVIDIA Jetson Linux Developer Guide on Docker storage. That's out of scope here, but easy to find.
